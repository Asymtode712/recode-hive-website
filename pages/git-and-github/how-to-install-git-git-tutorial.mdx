**Installing Git on Windows**
-----------------------------

**Step 1**:- Navigate to the official Git Website [https://git-scm.com/downloads](https://git-scm.com/downloads), if your system is Windows click on Windows icon, Git file with size nearly 48 mb will start downloading. Once download is completed open the **git.exe** file.

![Windoes downloading screen](https://recodehive.com/wp-content/uploads/2021/04/image-174.png)

**Step 2**:- After downloading, set it up for your PC.  Setting up is a easy way, Hope you can do the same.

![How to install Git? Git Tutorial](https://recodehive.com/wp-content/uploads/2021/04/image-176.png)

**Step 3**:- After that setup, you can see in your PC’s application list there are few new apps are added like ‘Git Bash,’ ‘Git GUI,’ ‘**Git CMD**.’ However, we will mostly use **Git Bash** for uploading our projects.

![Git bash opening ](https://recodehive.com/wp-content/uploads/2021/04/image-178.png)

Execute the below command to see your current version of git in Git CMD or windows command promt. Git Installation on Windows is completed.

git --version

![Checking version of git](https://recodehive.com/wp-content/uploads/2021/04/image-179.png)

**Installing Git on Mac**
-------------------------

There are four ways we can install Github on mac
1. Install Via [Homebrew](https://brew.sh/)
2. Xcode Approach
3. Binary Installer
4. Building from source

Here in this post, we will be using [homebrew](https://brew.sh/) to install, Go to the homebrew website, copy the install command, once copied go to the mac terminal copy-paste the code, Mac will ask you to enter the system password, click enter. It may take 2 to 3 minutes to install homebrew

![installing homebrew](https://recodehive.com/wp-content/uploads/2021/04/image-181.png)

Once Installed use the below command to check version of the brew on terminal
brew -v

Now you can install git on mac system by entering below command on terminal, In future if you want to update the version of git installed on mac or windows use the **git clone** [**https://github.com/git/git**](https://github.com/git/git ) command on terminal or git bash.

brew install git-gui
