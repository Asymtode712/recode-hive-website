### GETTING STARTED

Wondering how to increase the website speed? The [Hummingbird plugin](https://premium.wpmudev.org/project/wp-hummingbird/) can help you in this case. So before moving ahead I would suggest checking your current website speed on **[Google page Insight](https://developers.google.com/speed/pagespeed/insights/)**, so once after you install the plugin on WordPress check it after 48hrs.
