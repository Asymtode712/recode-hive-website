How to start with Android Studio?
---------------------------------

Android Studio is the first thing you should know before you start the journey as an android app developer, Just a heads-up, it’s literally an ocean to catch up with, so don’t lose focus, build skills one by one and learn it. Google provides different platforms as well languages to build the application like Flutter, Android Studio, Arctic Fox, Kotlin, Java, etc.

What is GitHub?PRO AccountPush, Clone, Pull

GitHub is a version control system that will help you to control the version of the project on a very large scale.

GitHub offers Premium account for use, Premium users get unlimited Github Repo time, Storage etc. To know more [check out](https://github.com/pricing) GitHub Pricing.

Pull – Helps to get the repository from remote system to your local system whereas push does the opposite, and clone makes a copy of the parent repo.

To access Github, Go to [**Github.com**](https://github.com/join)  **in a web browser.**

**What is latest version of Android Studio?**

**Can I revert back to older version of Android Studio from the new version?**

No, You cant simply switch back the versions from installed software, there is another way of doing this uninstall the latest version and download the existing stable version and install it in your system.
