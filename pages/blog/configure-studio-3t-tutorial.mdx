Connection Setup
----------------

**Step 1:** Double click on the connection which you set up earlier. This will open up the existing connection on the left side of the screen.

![](https://recodehive.com/wp-content/uploads/2021/07/image-93-1024x433.png)

**Step 2:** Create a new database (Right-click the connection name, i.e., ‘**Tweet localhost:27017**‘, the name will be different based on what name you gave for the connection while you setup then click ‘Add Database…’)

![](https://recodehive.com/wp-content/uploads/2021/07/image-94.png)

**Step 3:** Give the Database you want to be as this is the demo one, im going to give the name Practical\_1. Click on Okay once you done the setup.

![](https://recodehive.com/wp-content/uploads/2021/07/image-95.png)

**Step 4:** You can see the new database created along with few folders inside.

![](https://recodehive.com/wp-content/uploads/2021/07/image-96.png)

**Step 5:** Right click on the database and click on the import button to import the file to your system.

![](https://recodehive.com/wp-content/uploads/2021/07/image-97.png)

**Step 6:** Now i have a json file with me, to import the JSON file click on the first option and **click on configure**. Now the new screen will open with Query table.

[Download sample File](https://drive.google.com/file/d/1pXBz5aA03Um4ZXdG0sQIzvNFgVDOXJ8E/view?usp=sharing)

![](https://recodehive.com/wp-content/uploads/2021/07/image-98.png)

**Step 7:** Click on the + sign to import the Json file to the screen.

![](https://recodehive.com/wp-content/uploads/2021/08/image-22-1024x321.png)

**Step 8:** Click on the Run button, you will get a pop up window saying Confirm the import “In total, 1 import unit will be run” Click on okay and proceed.

![](https://recodehive.com/wp-content/uploads/2021/08/image-23.png)

**Step 9:** Double click on your sample Json file to see the Json code. We will look more into the UI explanation in the upcoming posts.

![](https://recodehive.com/wp-content/uploads/2021/08/image-24-1024x515.png)

**Step 10:** To write queries opent the Intellishell, this can be done by right clicko the sample Json file you imported. The code you see on the right side if executed will produce relevant output.

![](https://recodehive.com/wp-content/uploads/2021/08/image-25-1024x390.png)

**Step 11:** You can save the script by from the file menu. The script will be then available on the database folder created in the connection.

![](https://recodehive.com/wp-content/uploads/2021/08/image-26-1024x557.png)
