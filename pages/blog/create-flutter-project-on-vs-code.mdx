Create Flutter Project on VS Code
=================================

Tags: Android,flutter

**Author:** Sanjay KV

**Date:** Thu Aug 05 2021

Have you heard about Flutter or started working on it, If you are a beginner thinking about how to proceed further this is the best tutorial you can start with. Why preference to VS Code & why not [Android Studio?](https://recodehive.com/installing-flutter-on-mac-2021-update/)

VS Code IDE (Integrated Development Environment) is very simple and fast loading and execution of emulators which makes things easy for developers. So yes, try in VS Code, and let me know your feedback on the comment session.

#### PREREQUISITES

*   Must have a stable [Flutter version installed.](https://recodehive.com/how-to-install-flutter-in-mac-os/)
*   VS code Platform on your system

[**DOWNLOAD THE APP**](https://code.visualstudio.com/)

#### HOW TO BUILD 1ST PROJECT?

**Step 1:** I hope you are qualified with the prerequisites, Now open **VS code**. So you will get the below IDE. Go to **View -> Command Palette**. This is the place where you can create a new **Flutter project** with less Navigation.

![How do you use flutter in Visual Studio?](https://recodehive.com/wp-content/uploads/2020/06/Screenshot-2020-05-25-at-9.58.22-PM.png)

**Step 2:** Now it’s a solution for how to create a new flutter project? All you need to search Flutter in Command Palate. Now select a **new Flutter** option from there.

![](https://recodehive.com/wp-content/uploads/2020/06/Screenshot-2020-05-25-at-9.58.50-PM.png)

**Step 3:** Once you did that you need to create a new **Flutter project,** Type the name of the new Flutter project you want to create and press enter.

![](https://recodehive.com/wp-content/uploads/2020/06/Screenshot-2020-05-25-at-9.59.35-PM.png)

**Step 4**: Once you did that check the bottom panel for the notification. you will get the status, there will be a common issue if **Flutter Package** was not installed properly.

![](https://recodehive.com/wp-content/uploads/2020/06/Screenshot-2020-05-25-at-10.02.06-PM.png)

**Step 5:** You can see the **No Device** option if you want to run an emulator on your system. If already a device is connected to your system, then you can run on the system.

![](https://recodehive.com/wp-content/uploads/2020/06/Screenshot-2020-05-25-at-10.16.48-PM.png)

**Step 6:** Next there is an option to select the emulator you want to run. You have options like

*   Android ARMv7a Nougat
*   Android Accelerated Nougat
*   Custom Android emulator

![](https://recodehive.com/wp-content/uploads/2020/06/Screenshot-2020-05-25-at-10.17.01-PM.png)

**Step 7:** This is how the emulator looks like, Make sure you have at least **8GB of memory** on your system to run this smoothly. Can Turn off the Android emulator as highlighted below.

![](https://recodehive.com/wp-content/uploads/2020/06/Screenshot-2020-05-25-at-10.17.48-PM.png)
