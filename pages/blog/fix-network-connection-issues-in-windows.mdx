#### GETTING STARTED

This is the most common issue in the Windows system especially the old windows system. when you try to access the internet if you get some issue **Fix network connection issues in Windows**.

Try to troubleshoot as the system says mostly won’t work and from that perspective you can try the below steps.

![Fix You aren't connected to a network issue in windows](https://recodehive.com/wp-content/uploads/2020/06/1.jpg)

**Step 1:** Let’s try the initial troubleshooting methods, so if you getting the above network issue while connecting to the internet, the best troubleshooting method is to activate the **airplane mode** and disable it and try to reconnect it.

![Network connectivity issue in windows.](https://recodehive.com/wp-content/uploads/2020/06/02.jpg)

**Step 2:** Getting “Unable to connect to this network” Press **cmd+R** and open your command prompt, then type devmgmt.msc.

This is a shortcut to open [Device Manager](https://www.computerhope.com/issues/ch000833.htm) or another way to open Device Manager is to search in your start menu. Device Manager is the place where you can see the drivers installed in your system and modify those.

![Fix You aren't connected to a network issue open device manger](https://recodehive.com/wp-content/uploads/2020/06/2.jpg)

**Step 3:** Go to Network adapters, then find your Interwork driver, in my system, it is Intel(R) driver, Right-click and **disables it**. Once you do this you can see the wifi option is disabled in your taskbar. Now Right-click on the same and **Enable it**.

![Fix You aren't connected to a network issue in windows](https://recodehive.com/wp-content/uploads/2020/06/3.jpg)

**Step 5:** Sometimes the system will ask you to restart. So next time when you open the browser next time you aren’t connected to the network issue won’t come.
