SAP CPI API Management – CPI Tutorials
======================================

Tags: CPI,SAP CPI

**Author:** Sanjay KV

**Date:** Mon May 31 2021

In this post will look into how to test SAP CPI and usage of API in the cloud platform Integrations. You must have done an end-to-end integration flow before proceeding and have download the required software for testing like Postman or SOAP. If you have not done end-to-end flow, please refer to the [SAP CPI Tutorials](https://recodehive.com/sap-cpi/) to proceed further.

**Step 1:** Navigate to your SAP CPI Tenant and click on Service instance under Service, if you dont find this, click on the overview icon on the left-hand side pane the window will expand you will able to see this.

![](https://recodehive.com/wp-content/uploads/2020/12/image-85-1024x285.png)

**Step 2:** After this click on the Cloud integration trial this will be based on the name you gave while setting up the tenant. Head to view credentials and copy the credentials. This may need in the future.

**Step 3:** Once you did that Go to Design, Develop, and Manage API, in the SAP Cloud [Platform Integration Suite](https://be481c33trial.integrationsuite-trial.cfapps.us10.hana.ondemand.com/).

![](https://recodehive.com/wp-content/uploads/2020/12/image-86-1024x328.png)

**Step 4:** In the next screen click on the API screen under Quick Actions. This will redirect you an pop asking Credential URL which copied in the step 1.

![](https://recodehive.com/wp-content/uploads/2020/12/image-96-1024x346.png)

**Step 5:** Select the radio butto URL and paste the URL you copied. Now name the details as follows then click on create.

![](https://recodehive.com/wp-content/uploads/2020/12/image-97-1024x558.png)

**Step 6:** Here you need to add the resource, Click on Add and Resource details.

![](https://recodehive.com/wp-content/uploads/2020/12/image-98-1024x244.png)

![](https://recodehive.com/wp-content/uploads/2020/12/image-100-1024x426.png)

**Step 7:** Once you added all information click on Deploy, The API will be Deployed now. Make sure you delete all the tab expect post

![](https://recodehive.com/wp-content/uploads/2020/12/image-102-1024x316.png)

**Step 8:** Developer can click on edit and click on API designer to see the backend code.

![](https://recodehive.com/wp-content/uploads/2020/12/image-104-1024x408.png)

![](https://recodehive.com/wp-content/uploads/2020/12/image-105-1024x177.png)

**Step 9:** Now you can see the Request product details click on try out to execute the API Resource created.

![](https://recodehive.com/wp-content/uploads/2020/12/image-106-1024x353.png)

![](https://recodehive.com/wp-content/uploads/2020/12/image-107-1024x361.png)

#### Assigning Policy Template

**Step 10:** Click on Discover button and need to select standard Package provided by SAP as shown below.

![](https://recodehive.com/wp-content/uploads/2020/12/image-108-1024x347.png)

![](https://recodehive.com/wp-content/uploads/2020/12/image-109-1024x270.png)

**Step 11:** Go to Artifacts and click on copy under Actions. This will create a copy of the artifacts.

![](https://recodehive.com/wp-content/uploads/2020/12/image-110-1024x227.png)

**Step 12:** Click on the developr option from the API Portal.

![](https://recodehive.com/wp-content/uploads/2020/12/image-111-1024x262.png)

**Step 13:** Proceed with click on API from the list (Req Prod)

![](https://recodehive.com/wp-content/uploads/2020/12/image-112-1024x162.png)

**Step 14:** Click on Edit button and click on policies. What we need to do now is to apply the policies template.

![](https://recodehive.com/wp-content/uploads/2020/12/image-113-1024x232.png)

**Step 15:** Click on Apply the policy template.

![](https://recodehive.com/wp-content/uploads/2020/12/image-114-1024x291.png)

![](https://recodehive.com/wp-content/uploads/2020/12/image-115.png)

**Step 16:** Click on Policy Editor and click preFlow this will intialise the flow and provide the necessary details as highlighted below.

![](https://recodehive.com/wp-content/uploads/2020/12/image-116-1024x392.png)

![](https://recodehive.com/wp-content/uploads/2020/12/image-117-1024x722.png)

**Step 17:** Now you need to Authorize the token, click on this icon has highlighted below. Enter the url you copied before and click on the update button to proceed.

![](https://recodehive.com/wp-content/uploads/2020/12/image-118.png)

![](https://recodehive.com/wp-content/uploads/2020/12/image-119-1024x183.png)

**Step 18:** Click on try out and Execture the Resource.

![](https://recodehive.com/wp-content/uploads/2020/12/image-120-1024x311.png)
