Error! Not able to see Active Subscription in SAP CPI
=====================================================

Tags: CPI,SAP

**Author:** Sanjay KV

**Date:** Fri Jul 24 2020

Facing Issue while setting up Trial Account Tenant, I have faced this after the recent UI changes of CPI.

Let’s Check the Issue: Are you seeing the Active subscription has 0 and when you move to the subscription tab and see any available here.

![SAP CPI Subscription tenant setup](https://recodehive.com/wp-content/uploads/2020/07/GetImage-1.png)

**Step 2:** Go to Entitlements

![](https://recodehive.com/wp-content/uploads/2020/09/GetImage-3-1024x659.png)

**Step 3:** Then you will be in this screen and click on configure Entitlements.

![](https://recodehive.com/wp-content/uploads/2020/09/GetImage-4-1024x363.png)

**Step 4:** Click on Add Service plan now.

![](https://recodehive.com/wp-content/uploads/2020/09/GetImage-5-1024x293.png)

**Step 5:** Now select Process Integration & Integration Suite from the list.

![](https://recodehive.com/wp-content/uploads/2020/09/GetImage-6-1024x537.png)

**Step 6:** Now you can see under available plans see list appearing click on that and Add 2 service plan you see in below.

![](https://recodehive.com/wp-content/uploads/2020/09/GetImage-7-1024x564.png)

**Step 7:** Now you can see something similar to the below screen and give some time for processing and task to get complete

![](https://recodehive.com/wp-content/uploads/2020/09/GetImage-12-1024x172.png)

**Step 8:** Now go back to the tenant screen you can see 1 active subscription available now as marked.

![](https://recodehive.com/wp-content/uploads/2020/09/GetImage-13-1024x319.png)

**Step 9:** Now you have to click on Subscription and you can see the Integration suite which is not subscribed, next task is to subscribe that.

![](https://recodehive.com/wp-content/uploads/2020/09/GetImage-9-1.png)

![](blob:https://recodehive.com/97f76f50-0090-4542-975e-aab8b96b2ae8)

**Step 10:** Yaay! We have reached the final step and you could see the Active subscription now. Kindly go back to the previous post to continue. The link is given below.

![](https://recodehive.com/wp-content/uploads/2020/09/GetImage-11-1024x381.png)
