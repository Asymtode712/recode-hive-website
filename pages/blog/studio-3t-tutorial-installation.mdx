Features of Studio 3T
---------------------

*   Data on a field level
*   usage methods like scrambles, nulls, and substitutions
*   Transform your collection’s schema
*   in just a few clicks
*   Auto complete queries with a powerful mongo shell
*   Build aggregation queries stage by stage and check inputs and outputs at each step

Installation Procedure
----------------------

**Step 1:** Download the latest [Studio 3T version](https://studio3t.com/download/) from the download, the file size would be around **217 MB**. you need to give the below information as a need before downloading Studio 3T.

![](https://recodehive.com/wp-content/uploads/2021/07/image-86-1024x525.png)

**Step 2:** Extract the zip file, and run the file using administrator privileges as shown below.

![](https://recodehive.com/wp-content/uploads/2021/07/image-87-1024x408.png)

**Step 3:** Click on Next on this screen this is nothing but the welcome setup wizard.

![](https://recodehive.com/wp-content/uploads/2021/07/image-88.jpg)

**Step 4:** Select the path you need to download the Studio 3T version, if you don’t have enough space in the C drive recommended to use other drive folders. Then click on the **Next button.**

![](https://recodehive.com/wp-content/uploads/2021/07/image-88-1.jpg)

**Step 5:** That’s it the installation part is almost completed. Check the box to run the Studio 3T and **click on the finish** button.

![](https://recodehive.com/wp-content/uploads/2021/07/image-88-2.jpg)

**Step 6:** You have to accept the license agreement to move forward. Click on I agree and next.

![](https://recodehive.com/wp-content/uploads/2021/07/image-88.png)

**Step 7:** This is an important step check the radio button Use Studio 3T’s default password encryption and click on the next button to continue the installation.

![](https://recodehive.com/wp-content/uploads/2021/07/image-89-1024x595.png)

**Step 8:** Select this option based on your requirement, I would recommend a shell-centric environment. This can be changed later too.

![](https://recodehive.com/wp-content/uploads/2021/07/image-90-1024x537.jpg)

**Step 9:** The Installation of the Studio 3T is completed now. You will be getting the below screen asking for the connection details. If you are a first-time user need to set up the connection details here.

Connect to local MongoDB server (Click on ‘**Connect**‘ to create a new connection, then click ‘**New Connection**‘ in the pop-up window)

![](https://recodehive.com/wp-content/uploads/2021/07/image-90-1024x504.png)

**Step 10:** Click on connection name and give any random name and click on save to complete the connection setup.

![](https://recodehive.com/wp-content/uploads/2021/07/image-92.png)

**Step 11:** Now the user can see the connection they have set up in the Studio 3T, here in my case I have set up the Connection name Tweet and these will be having the following database. Double click on the connection name.

*   admin
*   config
*   local

![](https://recodehive.com/wp-content/uploads/2021/07/image-93-1024x433.png)
