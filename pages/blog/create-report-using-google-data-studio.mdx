When comes to Data Visualization there are already marketing leading tools exist like **[Power BI](https://powerbi.microsoft.com/en-us/), [Tableau](https://www.tableau.com/learn/training/20203), [Snowflakes,](https://www.snowflake.com/) Informatica.** The sad part if all are required to get the paid version to use the maximum productivity out of it. Some of the features of Google Data Studio are the following. The Data studio lives in the cloud, anyone can use and access it from the cloud. You can use a template or create one your own. It **can load data from Google Analytics, Google ads, CSV, etc.** Once you connect these above accounts data will be uploaded manually. Easy free customization visualization for your business needs. But the Development feature is limited, you may not able to perform the DAX kinda features of PowerBI in [Google Data Studio](https://www.brainlabsdigital.com/marketing-library/the-guide-to-data-studio/).

#### Create a report in Google Data Studio

To get started with Data Studio you need to sign up with a Google account.

**Step 1:** Go to [datastudio.google.com](https://datastudio.google.com) or search “Google Data Studio” in google and click on the first link. Now it’s time to **connect your Data Source to make Data set** through connectors can say Data Source to be Google Analytics. and connector is kinda Authentication which makes it possible to fetch the data and make the Data set.

So what you can do if you want to import data from another source that the connector doesn’t support. You can download it in CSV and upload it to Google Data Studio.

![Search result of Google Data Studio](https://recodehive.com/wp-content/uploads/2020/12/image-6.png)

**Step 2:** To create Report click on the **+ icon** and create a blank report or else expand the template Gallery and click on any existing template. For more details on the **Existing template,** you can refer to the related post below.

![Google Data Studio Interface](https://recodehive.com/wp-content/uploads/2020/12/image-7-1024x347.png)

#### Connecting Data Sources with Google Analytics, Youtube reports.

**Step 3:** Click Data Sources, and agree to the terms and conditions then select the country you live in on the next tab then at the end click the data source you want to connect. There are 18 Data connect source available. Select one and continue.

![Creating report in Data Studio](https://recodehive.com/wp-content/uploads/2020/12/image-8-1024x281.png)

**Step 4:** In my case, I’m using **Google Analytics** to connect with my Data Studio and going to make a report I want to see and share with my team.

![Establishing data Connection using Data Studio](https://recodehive.com/wp-content/uploads/2020/12/image-10-1024x356.png)

**Step 5:** Click on Authorize on next screen

**Step 6:** On the top, you might find a screen called **Data Freshness** where you can decide the time where the data update frequency from Google analytics to your window.

![Data Refres option in Data Studio](https://recodehive.com/wp-content/uploads/2020/12/image-12-1024x451.png)

**Step 7:** Now select the view and click on connect on top right and you can change the type of data structure from the sheet. Once you done that click on Create Report.

![Data Segregation & Data Filter in Google Data Studio](https://recodehive.com/wp-content/uploads/2020/12/image-11-1024x283.png)

**Step 8:** Click on create a report on the right side, you can select a theme from the right side. Give a **proper title**, then click on the add a chart button and click on the chart you want to import.

![Google Data Studio report creation screen](https://recodehive.com/wp-content/uploads/2020/12/image-13-1024x376.png)

**Step 9:** Similarly add the details like this, and try to make a dashboard with all necessary charts on the right side you add the metrics.

![Final report ](https://recodehive.com/wp-content/uploads/2020/12/image-14-1024x577.png)
