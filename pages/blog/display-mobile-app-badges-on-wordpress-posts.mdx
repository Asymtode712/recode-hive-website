#### What’s the relevance of app Badges?

So the Bloggers have to come up with something interactive to display rather than an image with a link. In that case, WP-Appbox comes into the picture, will show you how to make a beautiful representation of Mobile apps on WordPress posts.

You can download the plugin here the WordPress application is called WP-Appbox. For customization, you can refer to [this website](https://www.greengeeks.com/tutorials/article/create-badges-for-mobile-apps-in-wordpress/). #### Android App badge Settings

To display App from **Google Playstore** to appear like shown below with image and QR code, you have to copy-paste the code like shown below. Once you install the plugin then you can paste the below code anywhere on your post, it represents first the Appbox by default it comes with all the implementation, second is store name eg Google play, Appstore, etc. Whatever comes after the Id in the address bar goes as app-id.

     [appbox storename app-id] The above highlighted is App ID

The app ID you will get from the play store.

#### Google Chrome Badge Settings

Now let’s see how to display Mobile app badges on WordPress for chrome extensions PostsNavigate to the Chrome Webstore site, you can [click here](https://chrome.google.com/webstore/category/extensions?hl=en) to go to the site this is the place where you can see all the chrome extensions. Replace the shortcode with a **chromewebstore** and paste the application Id from the website address bar. For Microsoft universal store and Xbox apps you can use **windowsstore, also the badge can make** appear for **Firefox add-ons** and **F-Droid** ad-on.

    [appbox chromewebstore niloccemoadcdkdjlinkgdfekeahmflj?hl=en]
