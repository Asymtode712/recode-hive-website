### GETTING STARTED

Image icon. Images you represent the brand, so it’s equally important to view as it is in Android application, there are many ways to do it. In this post we are gonna helps you to Change the app icon in Android Studio

![](https://recodehive.com/wp-content/uploads/2021/05/image-32-1024x513.png)

**Step 2:** Once you open the image asset click on the path button as highlighted below, select the app you designed from your local directory. If you have the plan to publish the app on Playstore, make sure that the app icon is not a copyrighted image, because if it’s copyrighted Google will reject your app submission.

![](https://recodehive.com/wp-content/uploads/2021/05/image-33-1024x674.png)

**Step 3:** In the above picture you can see the app icon has been changed and there is inconsistency with the Full Bleed Layers app icon, need to change the background of the same, for this click on the background layer radio button and click on a color, if you know the color code it’s better, paste the color code and click on next button.

![](https://recodehive.com/wp-content/uploads/2021/05/image-34-1024x677.png)

**Step 4:** This is the last phase of icon change all the left are folder files need to be changed, once you click on the finish button it will get saved and Gradle sync will start. you can run the emulator and check the progress.

![](https://recodehive.com/wp-content/uploads/2021/05/image-35.png)

**Note:** There is an [alternative approach](https://medium.com/@ujikit/add-icons-to-the-android-application-from-android-studio-ide-cd1af9348749) to do the same, Open Android Studio -> Right-click on **res** and copy-paste the folder or icon

**Step 1:** Go to your desktop and copy the image: **Right-click -> copy file,** if you don’t have the icons or images available download the non copyrighted image from the internet. If you don’t have any icons designed don’t worry you can use this [](http://This post we are gonna helps you with this query in a very easy way to add icons or images in Android studio applications.)[website](https://icons8.com/icons/set/education) to build all kinda icons.

![](https://recodehive.com/wp-content/uploads/2020/06/Screenshot-2020-06-28-at-1.41.50-AM-1024x462.png)

**Step 2:** Now come to the Android studio, and go the drawable section and **right click** and **paste** over there. Now a box will pop up asking to change the Name of the file, If you wish to change the name you can change the name. Here I’m going to change the file name to icon.png.

![](https://recodehive.com/wp-content/uploads/2020/06/Screenshot-2020-06-28-at-1.38.19-AM.png)

**Step 3:** Now go to the respective manifest file you can see android icon: and hover to the icon name over there and run the application. That’s all you need to do.

![](https://recodehive.com/wp-content/uploads/2020/06/Screenshot-2020-06-28-at-1.38.05-AM.png)
