How To Install Flutter :
------------------------

**Step 1:** To install Flutter on windows go to this Link: [Flutter.io](https://docs.flutter.dev/get-started/install)

![How To Install Flutter :](https://recodehive.com/wp-content/uploads/2021/12/image.png)

**Step 2:** Click on the Download button highlighted the size will **be around 1 GB**, once downloaded Extract the file, once extracted the size will be around 2.3 GB. So make sure you have enough space in the system.

![Getting the Flutter SDK.](https://recodehive.com/wp-content/uploads/2021/12/image-1.png)

Setting up the Enviornment Variables in Windows
-----------------------------------------------

**Step 3:** Before I set up the environmental variable, I have moved the extracted files into the document folder in windows.

![](https://recodehive.com/wp-content/uploads/2021/12/image-3-1024x425.png)

**Step 4:** To set up the path search **environmental variable** in the windows search bar then click on **edit the system environment variable**. System properties will pop up here click on Advanced Tab and click on **environment variable in the** bottom right corner of the window.

**Step 5:** Double click on the path, there you will see a list of paths you set for the other applications as well, you need to **add a new path** by clicking on the new button.

![](https://recodehive.com/wp-content/uploads/2021/12/image-2.png)

**Step 6:** The below is the path file where the folder is extracted. You can copy the address from the address bar from where the file has been stored. In my case it’s Documents.

    C:\Users\sanja\OneDrive\Documents\flutter_windows_2.8.0-stable\flutter\bin

![setting up path variables](https://recodehive.com/wp-content/uploads/2021/12/image-4.png)

**Step 7:** Now we can use check everything is done properly using **flutter doctor**, open cmd, and run the command flutter doctor.

![Running Flutter doctor to check flutter installations](https://recodehive.com/wp-content/uploads/2021/12/image-7-1024x650.png)

**Step 8:** Install [Visual studio code.](https://code.visualstudio.com/) once the installation is done, open the command pallet in a flutter by clicking on the view button and typing install. Then click on the **first option to install the Extensions.**

![Install Extensions via command pallet in Windows](https://recodehive.com/wp-content/uploads/2021/12/image-5-1024x401.png)

**Step 9:** Now click on the first Install Button, that’s it you are ready to code in Flutter.

![How to Install Flutter on Windows](https://recodehive.com/wp-content/uploads/2021/12/image-6.png)

**Again run the flutter doctor in the command prompt** to see everything as in place. The next tutorial will deal with how to install Flutter on Android Studio.
